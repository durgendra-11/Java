## 📂 Repository Structure

```
/java-basics        -> Syntax, variables, data types
/control-statements -> if-else, loops, switch
/arrays             -> 1D & 2D arrays, problems
/strings            -> String handling & methods
/oops               -> Classes, objects, inheritance, polymorphism, abstraction
/collections        -> List, Set, Map, etc.
```
